# 贪吃蛇游戏移动端 PWA 改造设计

## 概述

将现有桌面贪吃蛇游戏改造为支持移动端的 PWA 应用，包含：
- PWA 支持（可安装、离线可玩）
- 触控体验优化（大按钮、全屏滑动、防误触）
- 移动端 UI 重设计（新布局、更舒适的操作体验）
- 桌面/移动共存（一套代码自适应）

## 文件结构

```
snake-game/
├── index.html          # 主文件（重写）
├── manifest.json       # PWA 配置（新增）
├── sw.js              # Service Worker（新增）
├── icons/             # 图标目录（新增）
│   ├── icon-192.png
│   └── icon-512.png
└── docs/plans/        # 设计文档
```

## PWA 功能

### Manifest.json 配置

```json
{
  "name": "贪吃蛇游戏",
  "short_name": "贪吃蛇",
  "description": "经典贪吃蛇手机游戏",
  "start_url": "./index.html",
  "display": "standalone",
  "background_color": "#667eea",
  "theme_color": "#764ba2",
  "orientation": "portrait",
  "icons": [
    { "src": "icons/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "icons/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ]
}
```

### Service Worker 功能

- 缓存 index.html
- 缓存游戏资源（无外部依赖）
- 离线可玩
- 安装后自动激活

### 安装体验

- 首次访问显示安装提示横幅
- 安装后显示应用图标
- 全屏运行（无浏览器地址栏）
- 启动画面显示游戏名称

### 图标设计

- 使用简单的蛇形图标
- 背景色与游戏主题一致（紫渐变）
- 提供 192px 和 512px 两种尺寸

## 触控体验优化

### 触控方式

| 方式 | 描述 | 优先级 |
|------|------|--------|
| 虚拟方向键 | 点击屏幕底部的方向按钮 | 主要方式 |
| 全屏滑动 | 在任意位置滑动改变方向 | 辅助方式 |
| 点击画布 | 点击画布区域快速开始游戏 | 便捷操作 |

### 滑动控制

- 滑动检测最小距离：30px
- 方向锁定：同一方向滑动不会反向
- 水平滑动距离大 → 左/右
- 垂直滑动距离大 → 上/下

### 防误触设计

- 滑动死区：小于 30px 的滑动不响应
- 方向锁定：向右时不能直接向左
- 触摸反馈：按钮按下时有缩放动画
- 延迟优化：使用 `touch-action: manipulation` 消除 300ms 点击延迟

### 移动端特殊功能

- 防止屏幕休眠：游戏进行时保持屏幕常亮（Wake Lock API）
- 横屏提示：检测到横屏时提示竖屏体验更好
- 震动反馈：吃到食物或游戏结束时轻微震动（可选）

## 技术实现

### CSS 架构（Mobile-First）

```css
/* 基础样式 = 移动端 */
.game-container { /* 移动端布局 */ }
.controls { /* 移动端控制 */ }

/* 桌面端覆盖 */
@media (min-width: 601px) {
  .game-container { /* 桌面布局 */ }
  .mobile-controls { display: none; }
}
```

### 关键改动清单

| 类别 | 改动项 |
|------|--------|
| HTML | 添加 PWA meta 标签、manifest 引用、优化布局结构 |
| CSS | Mobile-First 重写、新的响应式断点、更大的触控区域 |
| JS | Service Worker 注册、安装提示、Wake Lock API |
| 新增 | manifest.json、sw.js、图标文件 |

### 兼容性

- iOS Safari 11.4+
- Android Chrome 67+
- 桌面 Chrome/Firefox/Safari/Edge

### 不变的部分

- 游戏核心逻辑（蛇移动、碰撞检测、食物生成）
- 四种食物类型和效果
- 计分系统
