# 🐍 贪吃蛇游戏

一个经典的贪吃蛇游戏，使用 HTML5 Canvas 和 JavaScript 实现。

## 游戏特性

- 🎮 简单易玩的方向键控制
- 🎨 美观的渐变背景和动画效果
- 📊 实时得分显示
- ⏸️ 支持暂停/继续功能
- 🔄 一键重新开始

## 如何玩

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开始游戏"按钮
3. 使用方向键 (↑ ↓ ← →) 或 WASD 控制蛇的移动
4. 吃到红色食物得分，蛇身会变长
5. 避免撞到墙壁或自己的身体

## 控制方式

- `↑` 或 `W` - 向上移动
- `↓` 或 `S` - 向下移动
- `←` 或 `A` - 向左移动
- `→` 或 `D` - 向右移动
- 点击"暂停"按钮暂停游戏

## 技术栈

- HTML5 Canvas
- Vanilla JavaScript
- CSS3

## 在线演示

只需下载项目并在浏览器中打开 `index.html` 即可开始游戏！
